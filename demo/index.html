WanaKana Demo

IME Mode (auto)

Text in lower case converts to hiragana.

Text in UPPER CASE converts to katakana.

IME Mode (hiragana)

Regardless of case, text converts to hiragana.

IME Mode (katakana)

Regardless of case, text converts to katakana.

IME Mode (auto, leave trailing n's)

Type something that ends with n (for example, konban) then tab out of the input box.

The n will stay there as is. Now try the same in the other input boxes above: the n is converted to kana!
